WCA Group Editor v1.0 – Automated Competition Group Planner

This tool loads competition data directly from the WCA API and automatically
generates groups for first rounds. Fast competitors are placed in later groups
and automatically assigned as scramblers, while slower competitors are assigned
as runners and judges. The results are exported as a clean PDF with color-coded
staff roles (orange = scramblers, green = runners, blue = judges).

Features:
- Load any WCA competition by ID
- Automatic group creation based on rankings (first rounds only)
- Smart staff assignment (fast → scramblers, slow → judges/runners)
- Color-coded PDF export per event
- Multi-event support in one session

Note: Version 1.0 supports first rounds only.
      Version 2.0 with support for subsequent rounds is in development.
